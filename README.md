# Colossus Admin

App Kivy gerado automaticamente via GitHub Actions (Buildozer).  
Cada push na branch `master`/`main` dispara a compilação e gera o APK como artefato.

## Como baixar o APK
- Vá em **Actions → última execução → Artifacts → APK**.

## Estrutura
- `main.py`: app principal
- `buildozer.spec`: configuração do Buildozer
